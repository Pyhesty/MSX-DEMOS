 BANDWAGON: ALANKOMAAT
 =====================
 Released at Plutonium 2000.
 Entirely coded and assembled with a 
 genuine SVI-728 MSX.
 Requires VDP1, PSG, 64k RAM, 
 16k VRAM, MSX-DOS1.

 Type AL.BAT from the prompt to run.

 nf used skmsx-editor, gen80 for 
 compiling and Borland Pascal for 
 precalcs.

 The effects
 -----------
 All effects fullscreen, high res. and
 50 Hz except when stated otherwise.
 Observe that this demo uses some 
 pretty special (and pretty undocumen-
 ted) MSX1-VDP-features, so if you 
 think it doesn't run correctly on
 your MSX, drop me a mail.

 #1 / 3-way scroller :
  - 8-pixel char scroll

 #2 / butterflies:
  - 32 sprites

 #3 / wobbling grid
  - 25 Hz character effect

 #4 / pisa #1
  - character effect

 #5 / pisa #3
  - 25 Hz character effect

 #6 / sinus-shower
  - chars + 12 sprites

 #7 / sample
  - 8 kHz, 12-bit
  
 #8 / bumpmap
  - 25 Hz character effect + 
    48 sprites

 #9 / rotating cubes
  - 25 Hz character effect

 #10 / scrolling picture
  - char scroll

 #11 / spiky wheel
  - character effect

 #12 / ending pic
  - extra high res. (256x384)
    interlaced
